### Virtual environment

Creating virtual environment
```
python3 -m venv .venv
```

Activate the virtuael environment

MacOS/Linux
```
source .venv/bin/activate
```

Windows
```
.venv\Scripts\activate
```

Deactivating virtual environement

```
deactivate
```

### Install requirements

```
pip install -r requirements.txt
```
